# Fire-Detection-System


A real-time fire detection system using OpenCV that detects fire from live camera feeds and triggers alerts.

## Features
- Real-time fire detection using HSV color segmentation
- Flicker analysis to reduce false positives
- Audio alert system
- Web-based status display using Flask

## Technologies Used
- Python
- OpenCV
- Flask
- NumPy

## How to Run
```bash
pip install -r requirements.txt
python fire_detector.py
